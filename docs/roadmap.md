# Roadmap

The following `pi-mono` areas are intentionally out of scope for the initial Rust port:

- browser UI components and storage stack
- Slack bot integration
- GPU pod lifecycle management
- OAuth login flows
- extension runtime compatibility with TypeScript packages
- clipboard image handling and rich attachment ingestion
- share/export integrations

The first Mu release focuses on the core terminal coding-agent experience and the crates needed to support it cleanly in Rust.
